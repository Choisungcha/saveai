## SaveAI 금융 데이터 통합 서비스

### 1. 서비스 개요
SaveAI는 사용자의 금융 데이터를 통합하여 분석하고, 맞춤형 금융 상품 추천 및 지출 분석 서비스를 제공합니다. 본 서비스는 금융 데이터를 안전하게 처리하며, 사용자의 개인정보를 보호하기 위한 다양한 보안 조치를 적용하고 있습니다.

### 2. 주요 기능
- 금융 데이터 통합 및 분석
- 맞춤형 금융 상품 추천
- 지출 분석 및 절약 제안

### 3. 보안 및 개인정보 보호 조치
- **데이터 암호화**: AES-256 암호화 및 클라우드 HSM을 사용하여 민감 데이터를 보호.
- **접근 제어**: 민감 데이터에 대한 접근 권한을 제한.
- **감사 로그**: 모든 데이터 접근 및 조작 기록.
- **데이터 마스킹**: 계좌번호, 카드번호 등 민감 데이터의 마스킹 처리.

### 4. 법적 준수 상태
- 개인정보보호법(PDPA): 준수
- 개인정보 영향평가(PIA): 완료
- 금융보안원 가이드라인: 준수

### 5. 데이터 처리 현황
- **총 사용자 수**: 0명 (서비스 출시 전)
- **연동된 계좌 수**: 0개
- **처리된 데이터량**: 0GB
